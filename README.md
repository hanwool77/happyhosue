

# HappyHouse


- 공공API를 활용한 **부동산 거래 정보 제공**과 함께 관련 서비스 제공
- **회원 관리** 및 전용 서비스를 위한 기능, **아파트 거래 정보** 제공, **관심 매물 관리**를 기본으로 한 통합 웹 서비스
- 기본 기능을 바탕으로 한 QnA 및 공지사항 게시판 운영, 채널 톡 서비스 연계하여 사용자 친화 UX/UI로 **사용자 중점**
